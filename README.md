# Arcane Warrior

Videojuego 2D desarrollado en Unity como **Trabajo de Fin de Grado (TFG)** del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM).

El proyecto consiste en un juego de acción 2D con selección de personajes, sistema de combate, enemigos y gestión de rondas, enfocado en la implementación de lógica de juego y mecánicas básicas.

## Contexto del proyecto

- Tipo: Proyecto académico (TFG)
- Estudios: DAM (dentro de la doble titulación DAM + DAW)
- Desarrollo: Proyecto realizado en equipo (2 personas)
- Motor: Unity 2D
- Lenguaje: C#

## Mi rol en el proyecto

El proyecto se desarrolló de forma colaborativa, repartiendo las tareas entre ambos integrantes y apoyándonos mutuamente en la implementación y resolución de problemas.

Mi aportación principal se centró en:

- Sistema de **selección de personajes**
- Implementación del **sistema de vida y rondas**
- Desarrollo de **enemigos** y su comportamiento básico
- Creación y gestión de **menús**
- Integración de **sonido y música**
- Apoyo general en la lógica del juego y ajuste de mecánicas

## Funcionalidades principales

- Juego 2D de acción
- Selección de personajes
- Sistema de vida y rondas
- Enemigos con comportamiento básico
- Menús de juego
- Sonido integrado
- Lógica de juego desarrollada en C#

## Tecnologías utilizadas

- Unity 2D
- C#
- Git (control de versiones)

## Estado del proyecto

Proyecto **terminado** como entrega final del TFG.  
Actualmente no se encuentra en desarrollo activo.
